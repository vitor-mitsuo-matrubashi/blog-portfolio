# Financial Dashboard

A personal project to track and visualize investment performance.

## Features

- Real-time stock data fetching
- Portfolio performance tracking
- Interactive charts and graphs
- Historical data analysis

## Technical Details

This dashboard was built using Python and several key libraries:

- **Pandas** for data manipulation
- **Streamlit** for the web interface
- **Plotly** for interactive visualizations
- **yfinance** for market data

## What I Learned

Building this project taught me about:

1. Working with financial APIs
2. Data visualization best practices
3. Deploying Python applications
