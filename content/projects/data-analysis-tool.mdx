# Data Analysis CLI Tool

A simple but powerful command-line tool for quick data analysis.

## Features

- Load and inspect CSV files
- Generate basic statistics
- Create simple visualizations
- Export reports in multiple formats

## Usage

```bash
# Analyze a CSV file
data-analyzer analyze data.csv

# Generate a summary report
data-analyzer report data.csv --output report.html
```

## Why I Built This

Sometimes you just need to quickly check a dataset without opening a Jupyter notebook. This tool provides instant insights from the command line.
